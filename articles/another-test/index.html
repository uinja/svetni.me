Polyglot Panic at the Snippet Disco

Written by A dev

This post exists to prove two things:

  1. Syntax highlighting works.
  2. Developers will build a snack-themed status dashboard in any language if left unsupervised.

Theme styling is handled in the blog CSS. Bring your own opinions about tabs, spaces, and semicolons.

JavaScript

const moods = ['ship it', 'add tests', 'panic quietly'];

function releaseDecision(coffeeCups) {
  return coffeeCups > 2
    ? 'deploy at noon'
    : (moods[coffeeCups] ?? 'write docs');
}

console.log(releaseDecision(1));
