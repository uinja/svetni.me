Welcome to the Build Goblin's Lair

Written by A dev

Welcome to your new Fusuma blog, where posts are forged at dawn and deployed before coffee.

Repeat after me: this blog obeys me, not the other way around.

This sample ships with RSS, pagination, archive pages, and just enough chaos to be educational.

The directory structure looks like this:

├── config.json               <- site configuration
├── contents
│   ├── about.md
│   ├── archive.json
│   ├── articles              <– each article has its own directory
│   │   ├── another-test
│   │   │   └── index.md
│   │   ├── bamboo-cutter
│   │   │   ├── index.md
│   │   │   └── moonlit_patch_notes.jpg
│   │   ├── hello-world
│   │   │   └── index.md
│   │   ├── markdown-syntax
│   │   │   └── index.md
│   │   └── red-herring
│   │       ├── ripe_banana.png
│   │       └── index.md
│   ├── authors               <- author metadata
│   │   ├── baker.json
│   │   └── the-fusuma.json
│   ├── css
│   │   └── main.css
│   └── feed.json
├── plugins
│   └── paginator.coffee      <- paginator plugin
├── templates
│   ├── archive.tsx
│   ├── article.tsx
│   ├── author.tsx
│   ├── feed.tsx
│   ├── index.tsx
│   └── layout.tsx
└── views
    └── articles.coffee       <- view that lists articles

Articles are sorted by date and 3 are shown per page by default. You can tune that in config.json.

A typical article frontmatter

---
title: We Deployed During Snack O'Clock
author: captain-commit
date: 2012-12-12 16:59
---

This paragraph becomes the intro on index pages.

## A h2, hr or <span class="more"> marks where the intro cuts off

Release notes:

- Feature works on my machine and two lucky laptops.
- Tests pass unless Mercury is in retrograde.
- Team morale up 14% after emergency cookies.

Links to other content are resolved automatically.

For example a link to ../bamboo-cutter/index.md resolves to /articles/bamboo-cutter/.

Now ship something fun. The first rule of demo blogs is simple: if a post doesn't make you grin, add one more ridiculous subtitle.
