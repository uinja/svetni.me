MCP (Model Context Protocol)

MCP (съкращение от Model Context Protocol) е отворен стандарт за свързване на ИИ модели с външни данни и инструменти. Той позволява на приложения като ChatGPT и Claude да взаимодействат със софтуерни екосистеми по сигурен и унифициран начин.
