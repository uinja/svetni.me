Xint Code

Xint Code е платформа на компанията Theori, която използва изкуствен интелект за автоматизирано откриване на уязвимости в сорс кода. Платформата е способна да идентифицира сложни логически грешки и пробиви в сигурността, които често убягват на традиционните инструменти за анализ.
