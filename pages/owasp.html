OWASP

Open Web Application Security Project (OWASP) е международна организация с нестопанска цел, фокусирана върху подобряването на сигурността на софтуера. Тя е известна със своите списъци „Top 10“, които дефинират най-критичните уязвимости за различни технологии. С навлизането на изкуствения интелект, OWASP разработва нови стандарти като „Agentic Skills Top 10“, насочени към сигурността на автономните агенти.
