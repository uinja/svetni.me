Cloud Security Alliance

Cloud Security Alliance (CSA) е световна организация с нестопанска цел, посветена на дефинирането и повишаването на информираността за най-добрите практики за осигуряване на сигурна облачна среда.

Дейност

CSA разработва стандарти, сертификации (като CCSK) и провежда изследвания в областта на киберсигурността. През 2026 г. организацията публикува ключови доклади относно сигурността на ИИ протоколи като MCP.

Свързани концепции

MCP (Model Context Protocol)
