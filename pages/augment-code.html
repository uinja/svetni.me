Augment Code

Augment Code е платформа за подпомагане на разработчици с изкуствен интелект. Тя предлага инструменти за автоматично допълване на код, дълбок анализ на големи кодови бази и оптимизиране на работните потоци при програмиране. Платформата следи и подпомага развитието на проекти в общността, включително инициативи като Open Design.
