Blender Python API (bpy)

Blender Python API (често наричано bpy) е мощен интерфейс, който позволява на разработчиците да контролират всяка функция на Blender чрез езика за програмиране Python. Този интерфейс е критичен за автоматизиране на сложни графични процеси, генериране на синтетични данни за обучение на ИИ модели (компютърно зрение) и създаване на персонализирани инструменти за художници. През 2026 г. Anthropic стана основен поддръжник на неговото развитие.
