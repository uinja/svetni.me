GraalVM

GraalVM е високопроизводителна виртуална машина (JDK), разработена от Oracle, която поддържа компилация преди изпълнение (Ahead-of-Time - AOT) за езици от екосистемата на Java. Тази компилация позволява преобразуването на Java приложения в самостоятелни native изпълними файлове (native images). GraalVM решава традиционни проблеми на Java в конзолната и облачната среди, като осигурява незабавно време за стартиране, минимален отпечатък върху системната памет и малки размери на дистрибуциите (около 10 MB за типичен конзолен инструмент), правейки Java конкурентна на езици като Go и Rust за CLI разработки.
