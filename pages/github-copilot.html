GitHub Copilot

GitHub Copilot е инструмент за подпомагане на програмирането, базиран на изкуствен интелект, разработен от GitHub в сътрудничество с OpenAI. Той предлага довършване на код, генериране на цели функции и помощ при писане на документация директно в средата за разработка (IDE). Услугата използва специализирани модели от фамилията GPT, обучени върху милиарди редове публично достъпен код.
