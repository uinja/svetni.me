Docker Desktop

Docker Desktop е графично приложение за macOS, Windows и Linux, което позволява на разработчиците лесно да изграждат и споделят контейнеризирани приложения и микроуслуги. То включва Docker Engine, Docker CLI клиент, Docker Compose, Kubernetes и други инструменти.

Платформата е оптимизирана за локална разработка и предлага интеграция с облачни услуги, разширени възможности за мрежова сигурност и графичен интерфейс за управление на контейнери, изображения и обеми. От версия 4.74 Docker Desktop включва и вградения ИИ агент Gordon.
