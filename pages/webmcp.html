WebMCP

WebMCP (Web Model Context Protocol) е предложен от Google отворен стандарт за интеграция на агенти с изкуствен интелект в уеб браузъра. Той представлява разширение на стандартния Model Context Protocol (MCP), разработено специално за работа в клиентска (client-side) среда.

Основни цели и предимства

WebMCP е създаден да замени традиционните методи за управление на страници от страна на ИИ агентите (като сваляне на DOM модела, Dom scraping и изчисляване на координати по снимки):

  • По-добра надеждност: Уеб сайтът декларира изрично своите функции, което предотвратява грешки при промени в оформлението (layout shifts).
  • Намален разход на ресурси: Използването на структурирани инструменти вместо пълния DOM модел спестява до 90% от LLM токените.
  • Възможност за видимо изпълнение: Агентите задействат функциите директно на страницата, позволявайки на потребителя да наблюдава процеса.

Интерфейси за разработчици

  • Declarative API (Декларативен): Позволява анотиране на стандартни HTML форми с атрибути за лесно откриване от браузърния агент.
  • Imperative API (Императивен): Използва JavaScript и JSON схеми за описване и изпълнение на сложни уеб инструменти в браузъра.
