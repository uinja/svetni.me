Git Worktree

Git Worktree е функционалност на системата за контрол на версиите Git, която позволява на разработчиците да поддържат множество активни работни директории (чекаути), свързани с едно и също локално хранилище. Вместо да се клонира хранилището напълно или да се превключват клонове в единствена папка (което изисква запазване на промените чрез stash), Git Worktree позволява работа по различни клонове едновременно в напълно изолирани директории.

Тази технология е фундаментален градивен елемент за съвременните среди за автономно програмиране. Инструменти като Claude Code, Cursor и Codex използват паралелни Git среди, базирани на worktrees, за да позволят на софтуерните агенти да работят по различни задачи едновременно, без да влизат в конфликт с промените на потребителя или на други паралелно работещи агенти.
